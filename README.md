# RevealPass - Password Viewer & Toggle Utility

A secure, high-end Chrome browser extension (Manifest V3) that enables users to toggle the visibility of password inputs on any webpage. It uses a modern glassmorphic dashboard control panel and separates secrets and environment configs from the codebase.

## 🚀 Key Features

* **Global Enable/Disable Switch**: Easily turn the toggler icons on or off (Defaults to **OFF** for security).
* **DOM Injection (MutationObserver)**: Works dynamically on Single Page Applications (SPAs) like React, Angular, or Vue where login forms are rendered dynamically.
* **Security First**: Runs 100% locally in the browser context with **no external server calls**. Your password data never leaves your machine.
* **Separated Secrets**: Loaded from a gitignored `.env` file via an ES module loader.

## 🛠️ Setup & Installation

### 1. Generate Configuration Secrets
First, ensure you have Node.js installed. In this project directory, run:
```bash
node scripts/build-config.js
```
This parses the key-values in `.env` and generates the gitignored file `config/secrets.js` which is loaded at runtime.

### 2. (Optional) Run Python Virtual Environment
A Python environment is initialized in `venv/`. If you want to customize the generated icons or run other tools:
1. Activate virtual environment:
   * **Windows**: `venv\Scripts\activate`
   * **macOS/Linux**: `source venv/bin/activate`
2. Install pip dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. To regenerate icons:
   ```bash
   python scripts/generate_icons.py
   ```

### 3. Load the Extension in Chrome/Edge
1. Open Google Chrome or Microsoft Edge and navigate to: `chrome://extensions` (or `edge://extensions`).
2. Toggle the **Developer mode** switch (usually top right).
3. Click the **Load unpacked** button (top left).
4. Select the `password-viewer-extension` folder (the workspace root directory `m:\extensions`).
5. Pin the extension to your toolbar to access it quickly!

## 🧪 Testing the Extension

We have provided a test form environment:
1. Open `test-login-page.html` in your Chrome browser.
2. Click the **RevealPass** extension icon in your browser toolbar.
3. Turn the switch **ON**:
   - You will see a small slate eye-icon appear inside the right edge of password inputs.
   - Click the eye-icon to reveal/hide the typed text.
   - Click the **Injected Password Field** button to dynamically load a password input. The extension will automatically detect it and attach the icon.
4. Turn the switch **OFF**:
   - The eye-icons will be removed instantly.
   - Any currently revealed passwords are automatically hidden again.
